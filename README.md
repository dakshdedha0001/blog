# Blog Backend

This is a simple backend for a blogging platform made using Node.js, Express.js, and MongoDB.

Users can:

- Register and login
- Create blog posts
- Edit their own posts
- Delete their own posts

---

## 🛠️ Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- JSON Web Token (JWT)
- bcryptjs
- dotenv
- cors
